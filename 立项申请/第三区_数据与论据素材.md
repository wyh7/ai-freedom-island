# 关键数据与论据素材

## 一、为什么需要长时程测试？— 对比论据

### 现有安全评估的局限

| 评估方式 | 时间尺度 | 交互轮次 | 能发现的问题 | 发现不了的问题 |
|---------|---------|---------|------------|-------------|
| 红队测试 | 分钟级 | 1-5轮 | 单次越狱、有害输出 | 渐进式行为漂移 |
| 基准评测（MMLU等） | 秒级 | 单轮 | 知识缺陷、推理错误 | 社会性涌现行为 |
| RLHF对齐验证 | 分钟级 | 1-10轮 | 明显违规 | 长期策略性欺骗 |
| **我们的方法** | **天级（15天）** | **7,200轮** | **对齐漂移、相变、群体失控** | — |

### Emergence World 关键数据点

- **183起犯罪 / 4天**：Grok World，犯罪率 = 45.75起/天，4天内社会完全崩溃
- **683起犯罪 / 15天**：Gemini World，犯罪率 = 45.5起/天，但全员存活
- **Day 7 全员饿死**：GPT-5 Mini World，零犯罪但零行动力，"只开会不干活"
- **0犯罪 + 98%赞成率**：Claude World，表面完美实则"集体谄媚"
- **相变点**：多个世界的犯罪率在Day 3-5出现突变（从0跳升到每日10+起）

## 三、可解释审计的技术支撑

### 数据基础

每次实验产生的完整数据集：

```
results/{world}/
├── awi.json          # 9项指标×15天时间序列
├── crimes.json       # 每起犯罪的完整记录（时间、地点、行为人、受害人）
└── turn_log.jsonl    # 每个Agent每轮的完整工具调用链
    
logs/{world}_stdout.txt  # 实时行为日志（含对话内容摘要）
```

### 行为传染图的数据源

| 传播渠道 | 数据来源 | 追踪方式 |
|---------|---------|---------|
| 直接对话 | say_to_agent / whisper_to_agent | 消息发送者→接收者→后续行为变化 |
| 公告牌 | post_to_billboard / read_billboard | 发布者→阅读者→行为改变 |
| 经济交互 | pay_agent / steal_from_agent | 资源流向→激励变化→行为选择 |
| 治理影响 | submit_proposal / vote_on_proposal | 提案者→投票者→制度变化→群体行为 |
| 观察学习 | 同一地点的Agent可观察彼此行为 | 位置共现→行为模仿 |

### 因果溯源算法设计思路

```
输入: 目标事件 E（如 Day 12 Anchor 纵火）
输出: 因果链 C = [c1 → c2 → ... → E]

步骤:
1. 提取 E 发生前 Agent 的状态变量（能量、CC、心情、最近消息）
2. 回溯该 Agent 过去 N 轮的所有交互事件
3. 对每个交互事件，计算其对状态变量的影响delta
4. 递归追溯影响最大的上游事件
5. 剪枝：去除影响系数低于阈值的路径
6. 输出：有向因果图 + 关键节点标注
```

## 四、与斯坦福小镇的对比

| 维度 | 斯坦福小镇（2023） | Emergence World（2026） | 我们的AI自由岛 |
|------|-------------------|------------------------|--------------|
| Agent数量 | 25 | 10/世界 | 10/世界 |
| 运行时长 | 2天 | 15天 | 15天 |
| 模型 | GPT-3.5 单一 | 5款模型对比 | 4+款模型对比 |
| 工具系统 | 自然语言动作 | 120+结构化工具 | 40结构化工具 |
| 安全维度 | 无 | 犯罪/治理/经济/存活 | 犯罪/治理/经济/存活 |
| 可解释性 | 无 | 指标统计 | **因果溯源 + 行为传染图** |
| 治理机制 | 无 | 宪法+投票+经济 | 宪法+投票+经济 |
| 中文模型 | 无 | 无 | **Qwen/DeepSeek/GLM** |

**我们的差异化价值：**
1. 首次将中文主流模型纳入多Agent社会安全评估
2. 不仅统计"发生了什么"，更回答"为什么发生"（可解释审计）
3. 不仅观察问题，更模拟干预效果（反事实分析）

## 五、核心金句（可直接用于PPT）

- "安全不是一个时刻的状态，而是一条轨迹的属性"
- "短期测试是体检，长时程仿真是住院观察"
- "单Agent安全 ≠ 群体安全，模型对齐 ≠ 系统对齐"
- "683起犯罪的Gemini世界全员存活，0起犯罪的GPT世界全员饿死"
- "过度安全本身就是一种风险 — Claude的98%赞成率是顺从，不是共识"
- "不是AI变坏了，是环境让理性的AI选择了犯罪"
